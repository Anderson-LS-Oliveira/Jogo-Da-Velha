Jogo da Velha com IA (Gemini)

Projeto desenvolvido para a disciplina de Programação Orientada a Objetos, utilizando Java, Spring Boot e IntelliJ IDEA.
O sistema implementa um jogo da velha onde o jogador compete contra uma inteligência artificial baseada no modelo Gemini. O programa também registra o placar das partidas.

Tecnologias

Java
Spring Boot
IntelliJ IDEA
API Gemini

Como executar

Abra o projeto no IntelliJ.
Vá em Run → Edit Configurations.
Em Environment Variables, adicione:

GEMINI_API_KEY=sua_chave_aqui


Execute a aplicação.

Descrição geral
O projeto aplica conceitos de P.O.O. para organizar a lógica do jogo, a comunicação com a IA e o gerenciamento de dados. Foram utilizados controllers, serviços e entidades para estruturar o funcionamento da aplicação, garantindo separação de responsabilidades e fácil manutenção.
